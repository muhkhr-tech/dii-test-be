Headers: Bearer <token>

Request Body :

```json
{
    "name": "dr. Andi Wijaya Kusuma",
	"username": "dokterandi"
}
```

```json
    "message": "Berhasil mengubah user",
    "user": {
        "username": "dokterandi",
        "name": "dr. Andi Wijaya Kusuma"

}
```
